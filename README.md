# Data Warehouse ETL Pipeline

## 📌 Overview
This project demonstrates a simple ETL (Extract, Transform, Load) pipeline using Python and Pandas.

## ⚙️ Workflow
1. Extract data from CSV files (customers, products, sales)
2. Transform data using joins and calculations
3. Load the final dataset into a new CSV file

## 🛠 Tech Stack
- Python
- Pandas

## 📂 Project Structure
- data/ → input & output CSV files
- main.py → ETL pipeline script
- scripts/ → reusable scripts (future)
- sql/ → SQL queries (future)

## 📊 Output
Final dataset: data/final_sales_data.csv
